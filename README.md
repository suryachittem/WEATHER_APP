# 🌦️ Weather App

A **simple, modern, and responsive Weather Application** built using **React.js** and **Material-UI**.  
This app fetches **real-time weather data** from a weather API and displays key details like temperature, humidity, wind speed, and current conditions with a clean and user-friendly UI.

🔗 **Live Demo:**  
👉 [🌦️ Weather App Live »](https://weather-kd7d5cmvr-suryas-projects-f7de5ba8.vercel.app)

---

## 🚀 Features

- 🌍 Search weather by **city name**
- 📡 **Real-time weather data** using API
- 🎨 **Modern UI** built with Material-UI (MUI)
- 📱 **Fully responsive** for mobile & desktop
- 🌡️ Displays:
  - Temperature
  - Humidity
  - Pressure
  - Wind Speed
- ☁️ **Dynamic weather icons** for better visualization

### ⚙️ Installation & Setup

2️⃣ Navigate to project folder

cd weather-app

3️⃣ Install dependencies

npm install

4️⃣ Create .env file and add your API key

VITE_WEATHER_API_KEY=your_api_key_here

5️⃣ Run the app

npm run dev
